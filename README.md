# 📋 To-Do API

## 💻 Sobre o projeto  
To-Do API é uma aplicação back-end desenvolvida para gerenciamento de tarefas. A API permite que usuários autenticados criem, visualizem, atualizem e removam tarefas, garantindo segurança dos dados através de autenticação com JWT.

---

## ⚙️ Funcionalidades  
- Cadastro de usuários  
- Autenticação com JWT  
- CRUD de tarefas  
- Marcar tarefas como concluídas  
- Listagem de tarefas concluídas ou finalizadas  

---

## 🛠 Tecnologias  

As seguintes tecnologias foram utilizadas no desenvolvimento da API REST do projeto:

- Java 17+  
- Spring Boot 3  
- Spring Security  
- JWT (JSON Web Token)  
- Maven  
- MySQL  
- JPA / Hibernate  
- Flyway  
- Lombok
